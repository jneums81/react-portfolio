# React Portfolio

## Description
This application is a portfolio website that was made using react and it allows you to access different parts of the portfolio with the nav bar. The nav bar includes an about me section, a portfolio section, a page where someone can put your contact information, and a page with a link to the resume. Along with that, the footer contains links to Github and LinkedIn.
